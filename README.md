## 🎯 关于本指南

本指南是对 [Claude Code](https://github.com/anthropics/claude-code) 开源项目的深度解析，旨在帮助开发者：

- 🔰 **快速上手** Claude Code 的使用
- 🔧 **深入理解** 各个模块的工作原理
- 🛠️ **开发** 自定义插件、命令、代理和技能
- 🔬 **修改和扩展** 源码以满足特定需求
- 🤝 **贡献** 代码到 Claude Code 社区

## 🎓 学习路径

### 入门级（第1天）
学习模块 01-03，掌握基础使用：
- 理解 Claude Code 是什么
- 掌握插件安装和使用
- 学会创建自定义命令

### 中级（第2天）
学习模块 04-07，掌握高级功能：
- 理解代理和技能系统
- 掌握钩子自动化
- 学会集成外部服务

### 专家级（第3天）
学习模块 08-12，深入源码：
- 理解配置和上下文管理
- 掌握Git和终端集成
- 理解安全机制实现

详细学习计划请查看 [学习路径指南](./LEARNING_PATH.md)。
